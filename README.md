# Latres_123230018_ImamKhusain

## Latres adalah aplikasi mobile berbasis Flutter yang digunakan untuk menampilkan daftar TV Show menggunakan API TVMaze. Aplikasi ini memiliki fitur login, menampilkan detail show, menambahkan favorite show menggunakan Hive Database, serta profile pengguna dengan implementasi state management GetX dan konsep MVC.

---

# Features

- Login Authentication menggunakan SharedPreferences
- Menampilkan daftar TV Show dari API TVMaze
- Detail TV Show
- Favorite TV Show menggunakan Hive Database
- Profile User
- Bottom Navigation Bar
- State Management GetX
- MVC Architecture

---

# Technologies Used

- Flutter
- Dart
- GetX
- Hive Database
- SharedPreferences
- HTTP Package
- TVMaze API

## Steps to Clone and Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/username/Latres_123230018_ImamKhusain.git
```

### 2. Open Project Folder

```bash
cd Latres_123230018_ImamKhusain
```

### 3. Install Dependencies

```bash
flutter pub get
```

If needed, configure your Android device or emulator, then run the app:

```bash
flutter run
```

### 4. Run Application

```bash
flutter run

```

### 5. API Reference

This project uses public API from:
```bash
https://api.tvmaze.com

```
